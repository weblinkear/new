## 雪落无声，只是我已白了头

>除夕夜，她敲开我的门，递来一盒饺子。
>“我妈让我送来的。”
>我们做了十年邻居，这是她第一次主动找我。
>我正要道谢，她却突然哭了：
>“其实……饺子是我自己包的。”
>“我知道你每年都一个人过年。”
>饺子盒的底部，压着一张泛黄的作文纸——
>那是十五年前，我代写的情书，署名是她的名字。
>而收信人，是我。

---

雪是从傍晚开始下的，起初只是细碎的盐粒，敲在窗上簌簌轻响，后来便成了扯絮般的鹅毛，沉沉地覆下来，把窗外零星的灯火都捂得黯淡了。屋里只开了一盏台灯，光晕黄黄的，拢着书桌上摊开的旧卷子，也拢着这一室的清冷。楼上楼下隐约传来团聚的喧闹，电视声、笑声、杯盘轻碰声，隔着楼板与墙壁，闷闷的，像另一个世界传来的潮汐。我这里，是寂静的滩涂。

又一年的除夕。

我起身去烧水，铝壶坐在炉子上，发出空洞的、逐渐尖锐的呜咽。正要关上厨房的窗，目光却不由自主地飘向对面三楼那扇熟悉的窗户。灯亮着，暖融融的光，透过米白色的窗帘，晕开一小团毛茸茸的桔色。窗台上那盆绿萝，似乎比夏天时茂盛了许多，垂下长长的藤蔓，在玻璃上印下疏淡的影子。没什么特别的动静，和过去十年里大部分夜晚一样。我和她，就在这样相距不过二十米的两个格子里，守着各自的光，连偶尔在楼道碰见，也只是点点头，便侧身让过。

水“咕嘟咕嘟”地开了。我关了火，那呜咽声戛然而止，寂静重新涌上来，裹得更紧。就在这时，门被敲响了。

很轻，迟疑的两下，顿了顿，又是两下。在这爆竹声都稀落下去的雪夜里，清晰得有些突兀。

我愣了一下，才走过去开门。楼道里的声控灯大概坏了，没亮，只有屋里溢出的那点光，勾勒出门外人的轮廓。是她。穿着一件浅色的羽绒服，没戴帽子，头发上、肩头，落着未化的雪花，晶莹的，被昏光一照，像是撒了一层细碎的糖霜。脸颊和鼻尖冻得微微发红，眼神有些躲闪，手里捧着一个白色的保鲜盒。

“那个……”她的声音比敲门声更轻，几乎要散在楼道穿堂的冷风里，“我妈……让我送点饺子过来。”

我一时没反应过来，只是站着。十年了，我们隔着一道墙，做了十年邻居，见面点头的交情。送饺子？还是除夕夜？

“哦……谢谢，太客气了。”我连忙侧身，“进来坐会儿？外面冷。”

她摇摇头，脚却钉在原地没动，只是把那个保鲜盒往我手里一递。盒子是温的，隔着一层塑料，能感觉到里面食物团簇的暖意。

“不用了，就……顺手。”她垂下眼，盯着自己沾了雪水的鞋尖。

我接过盒子，沉甸甸的。“真是麻烦你们了，还惦记着。代我谢谢阿姨。”

她没应声。楼道里很静，能听到雪落在外面防火梯上的沙沙声。就在我以为对话就此结束，她也该转身离开的时候，却看见她仍旧低着头，肩膀几不可察地颤了一下。

然后，我听见一声极轻微的吸气，像努力在压着什么。

“其实……”

她抬起头，眼眶毫无预兆地红了，蓄满了水光，就那么直直地看着我，那里面翻涌着太多我读不懂，或者说不敢去读懂的情绪。泪水终于滚落下来，划过她微红的脸颊。

“其实饺子……是我自己包的。”她声音哽咽，带着哭腔，却异常清晰，每个字都像用尽了力气，“我妈根本不知道。”

我僵住，手里捧着的盒子似乎突然变得滚烫。

“我知道，”她吸了吸鼻子，泪落得更急，却不再躲闪我的目光，“我知道你每年……都是一个人过年。”

这句话像一颗小小的石子，投入我心底那潭以为早已枯寂的死水。微波泛起，撞在冰冷的石壁上，回荡着空洞的疼。我张了张嘴，却发不出任何声音。寒暄？道谢？此刻都显得无比苍白可笑。

她像是下定了决心，忽然伸出手，却不是擦泪，而是飞快地揭开我手中保鲜盒的盖子一角，又迅速合上。就在那一瞬间，借着屋里透出的光，我瞥见码得整整齐齐的饺子下面，似乎垫着什么东西，不是通常的保鲜膜。

一张纸。

边缘有些卷曲，颜色是陈旧的、不均匀的黄。

我的心猛地一跳。

她不再说话，只是看着我流泪，那眼神里有哀伤，有决绝，还有一种……燃烧了太久、终于只剩下灰烬的疲惫。

时间仿佛被这暖气不足的楼道冻住了。不知过了多久，她才极轻微地对我点了点头，像是完成了一个漫长而艰难的仪式，然后转过身，一步一步，走下黑暗的楼梯。脚步声很轻，很快就被雪夜吞没了。

我站在原地，直到那脚步声彻底消失，才像回过神来，踉跄着退进屋里，关上门，背靠着冰凉的门板，缓缓滑坐到地上。手里的保鲜盒放在脚边。

雪光映着窗，屋里比之前亮了些，那团桔色的灯光显得不合时宜的温暖。我盯着那个白色的盒子，看了很久，才伸出手，指尖有些颤抖，揭开了盖子。

饺子是常见的元宝形，捏得仔细，边褶均匀，白白胖胖地挤在一起，冒着微弱的热气，和一点点好闻的面粉与馅料混合的香味。

我伸出手，小心地，将那几个饺子拨到一边。

露出了盒底。

果然，垫在下面的，不是保鲜膜，而是一张折了好几折的纸。

纸很薄，边缘已经磨损得起了毛，那种老式作文纸的横格，蓝色线条都有些淡了。我屏住呼吸，用两根手指，极其缓慢地将它抽了出来。

纸被折得很小，很紧。我一点点将它展开。

首先看到的，是纸张右上角，一个用红笔画的很幼稚的五角星，和一个小小的、墨水已经褪成褐色的日期。那是……初中校庆的年份？算起来，十五年了。

然后，我看到了那熟悉的、工整中带着一丝稚嫩拘谨的字体。是我自己的字。十五年前，我替同桌的她，写的“情书”。

“……你走过操场的样子，像一阵春天的风。阳光落在你头发上，我觉得连尘埃都在跳舞。”

“每次发作业本，如果你的本子和我的挨在一起，我能高兴一整天。”

“我知道我很普通，可能你永远不会注意到我。没关系。只是想告诉你，有一个人，这样笨拙地、安静地，喜欢着你。”

信的末尾，是当时我模仿她的笔迹，练习了好几遍才写上去的、她的名字。而在那个名字下方，本该是空白的、留给“收信人”的地方——

我的目光死死定住。

那里，用另一种颜色的笔，墨水是深蓝色的，笔迹清秀却有力，是我后来在无数次交作业、楼道签名时见过的、属于她的笔迹。

写着我的名字。

收信人，是我。

时间轰然倒塌。

十五年前的阳光，毫无预兆地穿透此刻的雪夜，刺得我睁不开眼。那个总是低着头、说话细声细气的女孩；那个被我嘲笑“连情书都要人代笔”的下午；那张我随手递过去、像完成一件无聊作业般的信纸；她接过时飞红的脸颊和颤抖的手指……所有未被留意的细节，所有被岁月尘埃覆盖的碎片，在这一刻，被这张单薄的纸片悍然掀起，拼接成一副我从不敢想象、也无从想象的图景。

原来那阵春天的风，从未吹向别处。

原来那些尘埃的舞蹈，并非我一个人的独白。

原来那场我以为仅仅是少年时代一次尴尬帮忙的书写，对于她，是一次用尽全部勇气的交付。而我，像个最蹩脚的邮差，弄丢了最重要的信件，甚至忘记了自己也曾是那个……未被署名的寄件人。

十年邻居。点头之交。

她看着我一个人，年复一年。

她守着这个秘密，一年，又一年。直到今夜，用一盒温热的饺子，压着这张泛黄的信纸，像一个沉默的、最后的句点。

窗外的雪，还在落，无声无息，覆盖万物。

屋子里真静啊。静得我能听见自己血液冲刷耳膜的声音，能听见心脏在胸腔里迟缓而沉重地搏动，像一口即将枯竭的老钟。饺子微微的热气早已散尽，在冰冷的空气里凝成看不见的雾。那封信，那张薄薄的、脆弱的纸，此刻重逾千斤，压在我的掌心，也压在我的每一次呼吸上。

我抬起头，目光再次投向对面三楼那扇窗。

灯，不知何时已经熄了。

只有雪光，映着那盆绿萝垂落的、沉默的暗影。

原来，在我懵然不觉的岁月里，有人曾那样寂静地、固执地，爱了我这么久。久到那最初的心动，已凝成琥珀，封存在时光深处；久到那未送出的告白，被岁月风干，变成一张可以压在盒底、轻易不会碎裂的凭证。

而这十年的比邻，是守望，还是凌迟？

雪越下越大了。无声地，覆盖了来路，也模糊了归途。
